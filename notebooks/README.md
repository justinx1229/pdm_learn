# Notebooks

Active notebooks are grouped by workflow, but they are not the canonical
reproduction path. Run the command-line scripts documented in the top-level
`README.md` to regenerate manuscript tables and figures. Use notebooks for
interactive inspection and explanatory walkthroughs.

- `PPI/`: protein-protein interaction controls, feature generation, and downstream evaluation
- `oncogene/`: CCLE/DepMap trimming, oncogene ranking, and feature-conversion experiments
- `simulation/`: simulated association generation and model benchmarking

Historical notebooks that are not part of the reviewer-facing reproduction
workflow live under `archive/notebooks/`.
