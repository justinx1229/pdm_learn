# Archive

This directory holds provenance notebooks and historical analysis material that
are not part of the reviewer-facing reproduction workflow.

The active reproduction path is documented in the top-level `README.md` and uses
the notebooks under `notebooks/oncogene/`, `notebooks/PPI/`, and
`notebooks/simulation/`.

Archived notebooks may contain historical outputs, old exploratory cells, or
machine-specific paths from earlier development. Treat them as context, not as
entry points for reproducing the manuscript results.
